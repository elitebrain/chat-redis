### client file structure

```md
src
├─ components
├─ atomics - 기본 구성요소
├─ molecules - 기능적 구성요소
├─ organisms - 인터페이스
├─ templates - 기본 레이아웃
├─ pages - 각각의 페이지들
└─ PascalCase - 해당 페이지에서 사용되는 components
├─ utils - client 공용
└─ socket.js - socket.io-client 연결
├─ styles
└─ global.js - 전역 css
└─ index.js - 진입점
```
